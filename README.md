Comparative Analysis of Different Types of Prompting Patterns and Explanation with Various Test Scenarios

Aim
To test and compare how different prompting patterns respond to various prompts such as naïve prompts (broad or unstructured) and basic prompts (clear, refined, and structured) across multiple scenarios. The responses are analyzed based on quality, accuracy, and depth of the generated output.

AI Tools Required
• ChatGPT
• Internet Browser
• Computer System

Explanation
Prompt engineering is an important technique used to improve the performance of AI models such as ChatGPT. The quality of the response generated by an AI system depends greatly on how the prompt is written.
In this experiment, two different types of prompts are used:
Naïve Prompt
A naïve prompt is simple and less structured. It does not provide enough context or instructions to guide the AI model.
Example:
"Write about Artificial Intelligence."
Basic Prompt
A basic prompt is clear, structured, and provides specific instructions to guide the AI model.
Example:
"Explain Artificial Intelligence in simple terms with examples and describe its applications in modern technology."

Procedure
Define the two prompt types: Naïve Prompt and Basic Prompt.
Prepare multiple test scenarios such as creative writing, factual questions, summarization, and recommendations.
For each scenario, create both a naïve prompt and a basic prompt.
Input the naïve prompt into ChatGPT and record the generated response.
Input the corresponding basic prompt and record the response.
Compare both outputs based on quality, accuracy, and depth.
Analyze how prompt clarity affects the performance of the AI model.

Analysis
From the experiment, it is observed that basic prompts produce more structured and detailed responses compared to naïve prompts.
Key observations:
• Basic prompts provide clear instructions, helping the AI understand the task better.
• Structured prompts generate more accurate and detailed responses.
• Naïve prompts often produce short and less informative answers.
• In simple tasks, naïve prompts may still provide acceptable results.
Therefore, prompt clarity plays a significant role in improving quality, accuracy, and depth of AI-generated responses.


Result
The experiment successfully compared naïve prompts and basic prompts across different test scenarios using ChatGPT.
It was observed that basic prompts consistently generated more accurate, clear, and detailed responses compared to naïve prompts. This demonstrates that structured prompts significantly improve the performance of AI systems.
Thus, prompt engineering is an important technique for obtaining high-quality outputs from AI tools such as ChatGPT.
